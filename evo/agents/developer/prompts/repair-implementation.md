## 任务

1. **诊断**：使用 `ascendc-precision-debug` 诊断决策树定位问题根因
2. **修复**：在保留性能优化的前提下修复正确性
3. **编译验证**：确保编译通过

## 诊断参考

| 误差模式 | 可能原因 |
|---------|---------|
| 全零输出 | Pipeline 同步缺失、数据未搬入 |
| 大量 NaN/Inf | 精度溢出、除零 |
| 系统性偏差 | Cast RoundMode、算法错误 |
| 随机误差增大 | 数据竞争、未初始化内存 |
| Shape 不匹配 | InferShape 逻辑错误 |

## 约束

- 优先保留性能优化，只回退必要的部分
- 所有修改在候选目录中进行
- 编译通过后报告修复内容
